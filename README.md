# github_colab
Prueba de sincronización entre github y colab
## Paso a paso
1. Crear un archivo en Colab.

2. En "configuración" o "Settings" en la sección de GitHub autorizamos y sincronizamos con GitHub.
<img width="1810" height="846" alt="image" src="https://github.com/user-attachments/assets/56e8fffc-8909-499b-8c37-2c8e05de6ace" />


3. Después de dar acceso. (OPCIONAL) Se puede otorgar permiso para el acceso a repositorios privados.
<img width="870" height="653" alt="image" src="https://github.com/user-attachments/assets/9ff30ce5-4c03-4a68-8aef-4038204e51e9" />


4. Salimos a la pantalla principal y en el menú de "Archivo" o "File" se da click en "Guardar copia en GitHub"  o "Save a copy in GitHub".
<img width="1874" height="884" alt="Captura de pantalla 2026-04-17 092941" src="https://github.com/user-attachments/assets/8e48d89d-355c-4223-b89b-4bcd75070d71" />


5. Seleccionamos el repositoria al que queremos subir el archivo, seleccionando la rama **master**. Se da click en "Guardar" o "Save"

NOTA: El repositorio debe estar creado previamente, ya que, se pueden seleccionar **únicamente** repositorios existentes.
<img width="1667" height="845" alt="Captura de pantalla 2026-04-17 093336" src="https://github.com/user-attachments/assets/a1c1f050-26b0-46a7-b6b2-057a76b4e9b7" />

6. Finalmente obtenemos el GitHub con el archivo de Colab.
